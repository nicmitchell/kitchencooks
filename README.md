## Usage

1. Login using GitHub credentials
1. Seats show GitHub avatar of the occupant
1. Pick a seat that doesn't already have an avatar and click on it

## Getting Started

**Requires:** Node 0.10.x

**Within the root directory, run:**

    npm install
    bower install
    grunt
    nodemon
